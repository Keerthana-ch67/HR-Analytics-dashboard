An end-to-end HR Analytics dashboard built in Power BI to analysis employee attrition, workforce demographics, job satisfaction, and performance metrices using data modeling, DAX  and interactive data visualization.
# Key Highlights:

1. Attrition Analysis: Identified high‑risk departments and job roles by tracking turnover patterns.
2. Recruitment Metrics: Measured time‑to‑hire and cost‑per‑hire to evaluate recruitment efficiency.
3. Employee Engagement: Analyzed survey data to highlight drivers of satisfaction and retention.
4. Training Impact: Assessed training participation and post‑training performance improvements.
5. Interactive Dashboard: Developed executive‑ready visuals with drill‑down features for HR managers.
